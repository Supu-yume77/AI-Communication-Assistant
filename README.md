# 🤖 AI Communication Trainer

An AI-powered chatbot that analyzes sentence tone and helps improve professional communication.

## 🚀 Features
- Detects whether a sentence is polite or not
- Suggests improved professional versions
- Real-time AI-based response
- Clean and interactive UI

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI API: OpenRouter (GPT-based model)

## ⚙️ How It Works
1. User enters a sentence
2. Frontend sends request to Flask backend
3. Backend calls AI API
4. AI analyzes tone and returns response
5. Result is displayed to user
